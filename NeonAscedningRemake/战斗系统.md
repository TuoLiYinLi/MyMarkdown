		+ damage 基本攻击力

	+ 属性
		+ flag_valid 是否有效
		+ life_time 持续时间
		+ host 发射者
		+ data 拥有的投射物数据
	+ 方法
		+ offence(mob) 结算伤害 
	+ 信号
		+ on_hit_mob(mob) 当击中敌人时
		+ on_expired() 当失效时（）
		+ on_kill_mob 当击杀敌人时
